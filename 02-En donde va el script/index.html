Colocación del Script

El script puede ir en el HTML o como archivo externo.

0.- La etiqueta script

En HTML, el código JavaScript se inserta entre las etiquetas <script> javascriptCode </script>

Ejemplo

1.- El script puede ir en el html

Ya sea en head, en el body o en ambos, pero lo ideal es colocar el script al final del body, ya que la carga del script ralentiza el renderizado.

México

2.- El script puede ir en un archivo aparte.

Cuando el script va en un archivo aparte no se escriben las etiquetas script en ese archivo.
