Comentarios de Javascript

Los comentarios se pueden utilizar para explicar el código y hacerlo más legible.
También se pueden utilizar para evitar la ejecución al probar código alternativo.

1.- Comentarios de una sola línea

Los comentarios de una sola línea comienzan con //.
Cualquier texto entre // y el final de la línea será ignorado por JavaScript (no se ejecutará).
