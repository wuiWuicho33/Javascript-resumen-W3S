Identificadores

Los identificadores son nombres únicos con los que se identifican las variables, palabras clave, funciones y etiquetas.

Pueden ser nombres cortos (x,y) o nombres más descriptivos (edad, suma, volumen total).

Reglas para nombrar identificadores

Pueden comenzar con una letra.

JS distingue entre mayúsculas y minúsculas,
y e Y son identificadores distintos.

JavaScript no interpreta VAR o Var como la palabra clave var.
Pueden comenzar con los signos de ($) y (_):

Ambos son validos en JS ya que los interpreta como letras.

Signo de dólar $

Guión bajo (_)
Los nombres pueden contener:
  • Digitos: (1 o 10)
  • Guiones bajos: _
  • Signos de dolar: $
No se pueden usar las palabras reservadas Palabras clave de Javascript
No se permiten números como primer carácter. JS distingue los identificadores de los números.

Mayúsculas y minúsculas

Todos los identificadores distinguen entre mayúsculas y minúsculas.

Las variables lastName y lastname son dos variables diferentes.

Escritura de camello

Existen varias maneras de escribir un nombre de variable con varias palabras.

Guión (-) Guión bajo (_)
No están permitidos en JS ya que están reservados para restas.
first-name, last-name, master-card, inter-city. 		first_name, last_name, master_card, inter_city.
Camello superior (AB) Camello inferior (aB)
FirstName, LastName, MasterCard, InterCity. Los programadores de JS recomiendan comenzar con una letra minúscula y seguir con mayúsculas.
firstName, lastName, masterCard, interCity.
