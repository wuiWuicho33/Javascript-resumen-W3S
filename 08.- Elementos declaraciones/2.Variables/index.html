Variables

¿Que són las variables?

Las variables son contenedores que sirven para almacenar valores de datos.

Estas variables son usadas en expresiones como la siguiente:
(total = precio1 + precio2).

var, let y const

Let y const son palabras clave de la versión (ES6 - ECMAScript 2015)

  • var: declara una variable a nivel global.
  • let: declara una variable con alcance restringido.
  • const: declara un almacenador de datos con valor constante que no se puede reasignar.

Valor = indefinido

En los programas de computadora, las variables a menudo se declaran sin valor.
El valor puede ser algo que deba calcular o algo que se proporcione más adelante como una entrada del usuario.
Una variable declarada sin valor tendrá el valor undefined.
